# 🎮 Juego: Adivina el Número (con Funciones)

¡Bienvenido a mi proyecto del Bootcamp! Este es un pequeño juego interactivo desarrollado en Python donde el usuario debe adivinar un número secreto generado por un ordenador.

## 📝 Descripción
El programa utiliza **funciones** para organizar la lógica del juego:
* Generación de un número aleatorio.
* Validación de la entrada del usuario.
* Comparación de números y sistema de pistas (mayor/menor).
* Control de intentos.

## 🛠️ Tecnologías utilizadas
* **Lenguaje:** (Python)
* **Conceptos aplicados:** Funciones, Bucles (While/For), Condicionales e Interacción por consola.

## 🚀 Cómo jugar
1.  Clona este repositorio o descarga el archivo `adivina_el_numero_funciones`.
2.  Ejecuta el archivo en tu terminal o editor favorito.
3.  Sigue las instrucciones en pantalla e intenta adivinar el número en el menor número de intentos posible.
